# ZellWerk

Interaktive Übungsseite für Biologie (7. Klasse) zum Thema **Zelle**.

## Inhalte

- Pflanzenzelle und Tierzelle
- Bestandteile und Aufgaben
- Blattzellen und Fotosynthese
- Zellen der Sprossachse: Leitung von Wasser und Nährstoffen
- Zellteilung und Wachstum
- spielerische Interaktion mit Karten, Zuordnungsspiel und Quiz

## Lokal starten

Einfach `index.html` im Browser öffnen.

Optional mit kleinem Webserver:

```powershell
cd c:\github\cell
python -m http.server 8081
```

Dann im Browser öffnen:

- `http://localhost:8081`
